Here we have one billion texts, each associated with one or more of 23 labels (a person can belong to more than one group) and the task is to build a machine learning model that learns what kinds of texts belong to which label. If everything goes well, the model should be able to correctly predict labels of some new unlabled texts. This task is a classification task that belongs to the more general category of supervised learning. Supervised means that we provided the machine with the correct labels during training (in contrast to an unsupervised task like "clustering", for example, in which we don't have any labels). Here we used several machine learning algorithms including MultiNomial Naive Bayes classifier and Deep Neural Netwroks. Before we implement these models, we need to translate the text into numbers so the machine would understand them. This is a Feature Extraction process, in which we define which features should be either selected, built or engineered to feed into model as the best representation of our texts. For this Natural Language Processing task, we implemented a variety of encoding algorithms ranging from a simple Bag-of-Words model, to more complicated ones like Word2vec and BERT Model.
